Version 1.2.3 - top

December 2014

Keep a separate backup of your current website before applying the following update.

Changes/Updates:

  • Fixes a bug with the CAPTCHA refresh link.
  • Forms can now be used inside modals (popups).
  • Update of jQuery version 1.11.2

New Files:

Place the following new files inside your website's root.

  • /assets/js/jquery-1.11.2.min.js

Updated Files:

First keep a separate backup of your current version of these files. Then it is recommended to use a file comparison tool (ex: WinMerge, FileMerge, etc) to separately merge the updated files with your current version of the same files.

  • index.html and index_w_extra_comments.html
    • Added "clearfix" class to .modal-body
    • Changed references to jQuery 1.11.2 version (javascripts section)
  • /assets/css/styles.css
    • Updates to: all .form-style rules (Lines 1169 - 1350)
  • /assets/js/functions.js
    • Updated function: validate_and_submit_forms
    • New functions: reset_forms, reset_captcha
  • /assets/js/initialise-functions.js
    • Updated function: initialise_document_ready_functions ('On Opening Popups', 'On Closing Popups' sections)
  • js/functions.min.js - if you have made extra changes to functions.js before this update, you'll need to re-minify it again after applying this update, using a tool like Prepros

Version 1.2 - top

22nd June 2014

Changes/Updates:

  • New changelog page with more information about the TWILLI Air updates.
  • Update of jQuery script version from 1.10.2 to 1.11.1
  • Fixed problem with opening popups on IE8 (IE8 is still not supported).
  • Linking to a section inside a popup is now possible.
  • Minor optimisation to the contact form.
  • Added subtle intro reveal animation (WOW.js + Animate.css), which can be applied to any sections.
  • Improvements to Bootstrap Elements' styling.
  • Added Font Awesome 4.1.0 for more scalable icons.
